PictureSlider Last updated: 6-20-2016

Displays pictures that you can quickly view through by sliding!

Functions

  • Load All -- All your pictures will be loaded into PictureSlider so you don't have to keep opening images one by one.
  • Multiple Loading Methods -- Loading options to help load images better to suit those with processor or memory restrictions.
  • Different Viewings -- See which pictures were previous or are next, or see just one picture at a time.
Try it out: Download from github!
Github for source code

Requirements: Latest Java installed or Java Runtime Environment (preferred, older versions not tested).
